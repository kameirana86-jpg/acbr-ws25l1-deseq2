# Start R
To start R statistical Programming 
language type R in the command line of 
the R4.5WS conda environment

```{CMD}

R
```

# Install Packages for Bioconductor
Installing `BiocManager` package from CRAN (Comprehensive R Archival Network)
using `install.package()` function in R base. Please type the following,
```{R}
install.packages("BiocManager")
```
